\begin{frame}{\textbf{INTRODUCTION}}
    \begin{itemize}
        \item  We have made an app named \textit{"Regional Recipes"}.
        \item  This app is based on the famous recipes of India.
        \item And we also provide the making process of famous recipes of India.
    \end{itemize}
\end{frame}

\begin{frame}{\textbf{OVERVIEW OF THE APP}}
    \begin{itemize}
        \item This app consists of a starting page with a next button.
        \item After clicking next button, it will load into a Login/Sign Up page.
        \item If the user is new to the app, they can sign up, or if they already have an account, they can login by entering their credentials .
        \item Next page contains India map, the user can select a state to know about the famous recipes of that state.
     \end{itemize}
\end{frame}
\begin{frame}{ \textbf{OVERVIEW OF THE APP}}
    \begin{itemize}
        \item After selecting the state, it will load into the next page and display all the famous recipes of that state.
        \item According to the user selection, they can select any recipe from the list.
        \item When user selects the recipe from the list , then app displays the ingredients and making process of that recipe.
    \end{itemize}
\end{frame}
